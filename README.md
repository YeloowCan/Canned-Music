# Canned-Music

高仿网易云播放器

技术栈：React, Redux, React-Router, Sass, TypeScript

## 项目地址

[罐头音乐](https://cannedmusic.yellowcan.top/)

## API 接口

- [NeteaseCloudMusicApi](https://binaryify.github.io/NeteaseCloudMusicApi)

## 项目启动

- yarn install
- yarn run dev
- 在浏览器中访问：`http://localhost:5173`
